# Github Pages로 블로그 시작!   
<p class="caption">jekyll theme를 이용한 github blog 시작</p>
<br/>


카페24 호스팅이 만료된 김에 github에 페이지로 블로그를 만들 수 있다길래 github로 옮겨왔다.        
물론 처음에는 jekyll가 예전에 사용했던 테터툴즈나 워드프레스같은 설치형 블로그인줄 알고 시작한거지만... 쩝.       

하지만 jekyll는 블로그처럼 구성되어있는 static site에 가까웠다.       
게시글을 입력해서 자동으로 등록되는 뭐 그런 개념은 아닌 듯 하다. 

나는 감히 생각해볼 수 없는 프론트 개발이긴 그런식으로 프론트 개발을 해볼 수는 있겠지.       
포트폴리오 사이트는 정적인 사이트로 구성했고,        
글을 작성하는 블로그는 그냥 jekyll를 사용하기로 했다.      

아직 손봐야할 곳은 많다.
